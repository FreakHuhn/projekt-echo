# Project Echo

**A modular Discord bot powered by GPT – built as a learning journey.**

---

What is Project Echo?

Project Echo is a modular Discord bot that can:
- respond to chat commands (e.g. `!echo`, `!tip`, `!gamequiz`)
- run multiplayer quiz games
- send voice invites via DM
- remember users and context (with mood and history tracking)

---

Why this project?

I started this project after changing my career path due to health-related reasons. With *Project Echo*, I want to prove that I can learn and build real applications from scratch.

**Important note:**  
This bot would not exist without the extensive help of ChatGPT.  
About 95% of the code was written with AI support. My contribution was defining logic, integrating modules, debugging, and learning from the process.

---

Current status

This is a snapshot of an ongoing learning project – not yet finished, but fully functional in many areas.  
Feel free to check out the code, the docs in `/docs`, and follow the journey!

---

Files & Structure

- `main.py` – CLI version of the bot  
- `echo_bot.py` – Discord integration  
- `logic.py` – Command handling and session memory  
- `invite.py`, `quiz.py`, `gpt.py` – Skills / modules  
- `memory.json` – User memory (mood, history, state)  
- `/docs` – Design, roadmap, planning, error reports
